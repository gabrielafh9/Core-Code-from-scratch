## Reassignment

Which value does x have after execution of the following code?

``` Javascript
let x = 'Tic';
x = 'Tac';
x = 'Toe';
```
R// The value of x is "toe"

## Assign variables

``` Javascript
let x = 'Laurel';
let y = 'Hardy';
let z = y;
y = x;
x = z;
```
R// The value of x is "Hardy"
